# API-SERVER
node.js 搭建后台

## 快速开始

安装依赖

```bash
npm install
```

启动（确保已安装 nodemon）

```bash
npm run serve
```

## 测试账户

```
用户名: admin
密码：000000
```

## 接口实现步骤

1. 定义路由
2. 验证表单数据
3. 通过 SQL 查询数据库
4. 通过 SQL 更新
5. 返回处理结果

## 环境
- MySQL 5.7.31
- Node.js 14.18.2
